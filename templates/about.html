About Us

This app was made as a project for CS 361 at OSU.

This website can be used to find new or existing recipes to help you plan out your meals. The results will include a picture and a link to the full recipe, where you can find the ingredients and instructions to make an amazing dish!

In the main menu, you can find the following functionalities:

  1. Search by Category
    • Allows you to search by food categories, such as soups, salads, pastas, etc.
    • If the category you are looking for does not exist, you can manually enter your own category. The more specific your search is, the more likely the result will be as intended. An example would be instead of searching for "cookies", you can try searching for "chocolate chip cookies".
  2. Generate Random Recipe
    • Randomly generate a recipe for you. Return to main menu and continue to generate random recipes until you find one to your liking!
  3. Saved Recipes
    • View a list of recipes that you have saved. The links are clickable to allow for easy redirecting to the full recipe.

This recipe search was implemented using a microservice architecture to call and receive response from the Edamam API.
